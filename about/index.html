What's the Purpose?

Welcome to Cameron's Tech Journal! This site serves the purpose of sharing new ideas and will also act as a personal blog for disscussing various topics of interest. Much of the content on this site will be STEM oriented with a focus on technology and more specifically on computers and programming.

How was this Created?

This site is made up entirely of the languages html and css with small amounts of javascript included. It is currently hosted for free on Github Pages. The code editor of choice used to code this site was VS Code and Gitbash was used to commit and push updates. The languages needed were researched with help from Google's Gemini LLM.
